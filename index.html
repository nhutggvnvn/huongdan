I. Các Bước Cài Cẩu Hình DNS và Cài Game

Bước 1: Cài DNS chặn thu hồi chứng chỉ lưu ý Muốn app sống vĩnh viễn thì cài DNS này và không tắt DNS và khởi động lại máy là app dùng vĩnh viễn nhé

Tải Cấu Hình: Cấu Hình DNS ANTI VIP

( HD Cài Cấu Hình: Vào Cài Đặt Chung > Quản lí cấu hình & thiết bị > Cấu Hình Đã Tải Về > Cài Đặt > Nhập Mật Khẩu Thiết Bị > Bấm Cài đặt > Đóng )

( Chú ý: Truy cập thử ocsp.apple.com xem đã anti thành công chưa nếu thành công sẽ hiện không thể tải trang như hình phía dưới )

Tiếp Theo cài ngay Esign trực tiếp Nhớ lưu ý vào setting của Esign xem esign bạn có chứng chỉ tên gì thì add vào đúng chứng chỉ của nó nha hình bên dưới
Esign trực tiếp Appcenter

Bước 2: Nhập chứng chỉ

Mở ESign -> chọn tab File -> Bấm dấu 3 chấm ở góc phải -> chọn Nhập và chọn file .zip chứa chứng chỉ để thêm vào ESign.

Bấm file .zip vừa thêm và chọn Giải nén, tiếp theo mở thư mục vừa giải nén đến khi thấy file .p12 và .mobileprovision.

Bấm file .p12 và chọn Nhập chứng chỉ, sau đó điền mật khẩu (nếu có) và bấm OK.

Bước 3: Ký app

Mở ESign -> chọn tab File -> Bấm dấu 3 chấm ở góc phải -> chọn Nhập và chọn file .ipa cần ký để thêm vào ESign. Sau đó chọn Nhập.

Chọn tab Apps -> Bấm app cần ký -> Chọn Ký.

Trò chơi không có bản quyền chính thức, hãy cân nhắc kỹ trước khi tham gia.
Chơi quá 180 phút một ngày sẽ ảnh hưởng đến sức khỏe.
